Descripción

WebGL proporciona un modo inmediato de renderizado 3D optimizado para la web

Funciona a utilizando el elemento CANVAS de HTML5

Extiende las capacidades del JavaScript y permite generar graficos 3D en cualquier navegador compatible

Bajo nivel, sólo podemos "dibujar" primitivas

Ejecuta el código WebGL directamente en la tarjeta gráfica

GSLS, Shading Language

Está basado en OpenGL ES (Embedded Systems) por lo que sus sintaxis y funcionamiento resultan familiares
